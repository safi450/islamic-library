🔄 سير العمل النموذجي:

1. تحضير الكتاب:
   ┌─────────────────────────────────────┐
   │ • ملف PDF                           │
   │ • صورة الغلاف                       │
   └─────────────────────────────────────┘
                    ↓
2. رفع الملفات:
   ┌─────────────────────────────────────┐
   │ • PDF → books/                      │
   │ • صورة → images/covers/             │
   └─────────────────────────────────────┘
                    ↓
3. إضافة البيانات:
   ┌─────────────────────────────────────┐
   │ • افتح admin.html                   │
   │ • املأ النموذج                      │
   │ • احصل على الكود                   │
   └─────────────────────────────────────┘
                    ↓
4. تحديث قاعدة البيانات:
   ┌─────────────────────────────────────┐
   │ • افتح js/books-data.js             │
   │ • الصق الكود                        │
   │ • احفظ                              │
   └─────────────────────────────────────┘
                    ↓
5. التحقق:
   ┌─────────────────────────────────────┐
   │ • افتح index.html                   │
   │ • أعد التحميل (F5)                  │
   │ • تحقق من الكتاب                    │
   └─────────────────────────────────────┘
